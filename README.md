## Binary

> Some helpful utilities
>
> - Test k<sup>th</sup> bit is set: `n & (1 << k) != 0`
> - Set k<sup>th</sup> bit: `n |= (1 << k)`
> - Clear k<sup>th</sup> bit: `n &= ~(1 << k)`
> - Toggle k<sup>th</sup> bit: `n ^= (1 << k)`
> - Check power of 2: `n & n - 1 == 0`
> - Check if number is odd: `n & 1 != 0`
